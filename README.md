# To-Do List Village

To-Do List Village는 픽셀 아트 스타일로 디자인된 브라우저 기반 투두리스트 웹 애플리케이션입니다.
사용자는 할 일을 추가하고, 이를 완료하거나 실패함에 따라 시각적인 변화가 나타납니다. 단순한 목록 관리 도구를 넘어, 하루의 성취와 감정을 시각적으로 표현할 수 있도록 설계되었습니다.

## 주요 기능

할 일 추가 및 관리
말풍선 UI를 통해 직관적으로 할 일을 추가할 수 있습니다.

완료 시 하트 채워짐 및 꽃 심기
할 일을 완료하면 하트 아이콘이 채워지고, 정원에 꽃이 추가됩니다.

미완료 시 유령 아이콘으로 표시
기한 내 완료하지 못한 일은 유령 아이콘으로 표시되어 남겨집니다.

픽셀 아트 기반의 감성 디자인
배경, 캐릭터, 꽃, 구름, 말풍선 등 모든 UI는 게임풍 픽셀 아트 스타일로 제작되었습니다.

## 사용 방법

말풍선 입력창에 오늘의 할 일을 입력하고 추가 버튼을 클릭합니다.

완료한 할 일은 클릭하여 상태를 변경할 수 있습니다.

완료된 항목은 하트가 채워지고 정원에 꽃이 심어집니다.

실패한 항목은 유령 아이콘으로 변경됩니다.



## 사용 기술

HTML / CSS / JavaScript (Vanilla JS)

Google Fonts - Press Start 2P

GitHub Pages를 통한 정적 웹사이트 배포

## 제작 의도

매일의 할 일을 단순히 관리하는 것을 넘어, 시각적으로 성취감을 느낄 수 있는 도구를 만드는 것이 목적입니다.

감성적인 디자인과 직관적인 UI를 통해 사용자의 몰입도와 재미를 높이고자 했습니다.
